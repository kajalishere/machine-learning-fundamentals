# Model Evaluation

## Overview

This module demonstrates how to evaluate the performance of a machine learning classification model using standard evaluation metrics. The implementation uses the **Glass Identification Dataset** and a **Logistic Regression** classifier to illustrate how model predictions are assessed after training.

---

## Topics Covered

- Confusion Matrix
- Accuracy
- Precision
- Recall (Sensitivity)
- F1-Score
- True Positive Rate (TPR)
- True Negative Rate (TNR)
- False Positive Rate (FPR)
- False Negative Rate (FNR)
- Receiver Operating Characteristic (ROC) Curve
- Area Under the Curve (AUC)

---

## Dataset

This project uses the **Glass Identification Dataset**, a publicly available dataset. The multiclass dataset is converted into a binary classification problem to demonstrate binary classification evaluation metrics.

---

## Learning Outcomes

After completing this module, you will be able to:

- Build and interpret a Confusion Matrix.
- Calculate Accuracy, Precision, Recall, and F1-Score.
- Compute TPR, TNR, FPR, and FNR.
- Generate and interpret an ROC Curve.
- Calculate the Area Under the Curve (AUC).
- Evaluate the performance of a binary classification model.
