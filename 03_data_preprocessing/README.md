# Data Preprocessing

## Overview

This module demonstrates essential data preprocessing techniques used before training machine learning models. 

---

## Topics Covered

- Data Preprocessing
- Missing Values
- Missing Value Imputation
- Outlier Detection
- Interquartile Range (IQR) Method
- Z-Score Method
- Cross Validation
- Bias-Variance Trade-off

---

## Files

| File | Description |
|------|-------------|
| Data_Preprocessing.pdf | Presentation explaining data preprocessing concepts and techniques. |
| data_preprocessing.py | Python implementation for missing value handling and outlier detection. |

---

## Technologies Used

- Python
- Pandas
- NumPy

---

## Learning Outcomes

After completing this module, you will understand how to:

- Identify missing values in a dataset.
- Handle missing values using mean imputation.
- Detect outliers using the IQR method.
- Detect outliers using the Z-score method.
- Understand the importance of preprocessing before model training.

---

## Dataset

This demonstration uses the **Glass Identification Dataset**, a publicly available dataset commonly used for classification tasks.
